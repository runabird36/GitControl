# brach 정리 뿐 아닌, 내부 규약 정리
1. commit은 의미 단위로 할것
2. commit을 다 하고 push 만을 남겨두고 있을때, 중간에 추가 내용이 생각나도 해당 단계까지가 의미단위이면 진행하고 추가로 다시 commit할것
3. branch 할 타이밍을 잘 생각
4. branch로 분리하던 하나의 main branch를 하던, 공동 작업시, 하나의 파일을 동시 작업하지 않도록 업무 분리
